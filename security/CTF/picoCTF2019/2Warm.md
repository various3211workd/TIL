# 2Warm

## 問題
> Can you convert the number 42 (base 10) to binary (base 2)? 

## 解答

```shell
$ python                                                                                 Python 2.7.16+ (default, Jul  8 2019, 09:45:29)
[GCC 8.3.0] on linux2
Type "help", "copyright", "credits" or "license" for more information.
>>> bin(42)
'0b101010'
```
